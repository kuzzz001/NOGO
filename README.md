项目名称
不围棋（NoGo）AI 对抗程序

项目简介
本项目实现了一个基于 C++ 的不围棋（NoGo）博弈程序，用于在 Botzone 平台进行自动对弈。程序通过读取当前棋盘状态，计算并输出一个合法且尽可能优的落子位置。

不围棋是围棋的变体规则，其核心特点如下：

1. 棋盘大小为 9×9
2. 双方轮流落子，黑先
3. 不允许吃子（如果落子导致对方无气则直接判负）
4. 不允许自杀（落子后自身无气）
5. 不允许跳过回合（无合法落子即判负）

项目目标是在保证合法性的前提下，通过一定策略提升胜率。

运行方式

本地运行
使用 g++ 编译后，通过输入重定向进行测试。

Botzone 平台运行
将程序上传至 Botzone 平台，由平台负责对局调度。程序每次运行只需完成一次决策。

⸻

输入输出说明

输入为当前对局历史记录，包括双方所有已落子位置。程序根据输入重建棋盘。

输出为当前回合的落子坐标（x, y）。

⸻

核心功能

1. 棋盘状态维护
2. 连通块与气的计算
3. 合法落子判断（避免自杀和吃子）
4. 决策策略
