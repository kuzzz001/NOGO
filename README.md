项目名称
NoGo AI（不围棋对抗程序）
项目简介
本项目实现了一个基于 C++ 的 不围棋（NoGo）AI 博弈程序，用于在 Botzone 平台进行自动对弈。程序能够根据当前棋盘状态，在 1 秒时间限制内计算并输出最优（或近似最优）落子位置。
不围棋是围棋的变体规则：
  不能吃子（吃子直接判负）
  不能自杀（无气落子判负）
  不能 pass（无棋可走判负）
目标是：避免非法操作并迫使对方进入无合法落子的局面
运行方式
本地运行（简单交互）
g++ nogo.cpp -o nogo
./nogo < input.txt
Botzone 提交
  上传 .cpp 文件
  平台自动调用程序进行决策
  每次调用只需输出一步落子
输入输出格式
输入
k
x1_op y1_op
x1_my y1_my
...
xk_op yk_op
输出
x y
核心功能
  棋盘建模（9×9）
  合法落子判断（无气判断）
  禁止吃子检测
  AI 决策模块（策略/搜索）
  时间控制（防止超时）
